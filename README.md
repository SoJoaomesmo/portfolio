# 💻 Meu Portfólio Web

Um site pessoal desenvolvido como portfólio para apresentar meus projetos, habilidades e informações de contato. O projeto foi estruturado com foco na internacionalização (suporte a Português e Inglês) e em uma experiência de usuário fluida através de animações de entrada orientadas pelo scroll.

## ✨ Funcionalidades

- **Site Bilíngue:** Navegação perfeita entre os idiomas Português e Inglês, gerenciada localmente pelas rotas do site.
- **Animações de Scroll (View Timeline):** Utilização de propriedades avançadas e modernas do CSS (`animation-timeline: view()`) para criar efeitos de surgimento suaves à medida que o usuário rola a página.
- **Tema Escuro Moderno:** Interface projetada com um background em degradê escuro e design responsivo.
- **Estrutura Modular:** Código construído com componentes React, permitindo fácil escalabilidade e manutenção.

## 🛠️ Tecnologias Utilizadas

- **[React](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/):** Base do projeto para a construção da interface do usuário com tipagem estática.
- **[React Router DOM](https://reactrouter.com/):** Gerenciamento das rotas (`HashRouter`) para alternar entre as páginas de idioma.
- **[Bootstrap](https://getbootstrap.com/):** Framework CSS para garantir responsividade e componentes rápidos.
- **CSS3:** Estilização global avançada, incluindo _keyframes_ e manipulação de _scroll-behavior_.

## 🚀 Como Executar o Projeto Localmente

Siga os passos abaixo para rodar o projeto na sua máquina:

1. **Clone o repositório:**

   ```bash
   git clone [https://github.com/SoJoaomesmo/portfolio.git](https://github.com/SoJoaomesmo/portfolio.git)
   ```

2. **Navegue até o diretório do projeto:**

```bash
cd portfolio
```

Instale as dependências:
Você pode usar npm, yarn ou pnpm (dependendo do que você utilizou para criar o projeto).

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

O projeto geralmente ficará disponível em http://localhost:5173 ou http://localhost:3000.
